# Package R - BUT Sciences des Données 
Ce package R a été développé par des étudiants du BUT Sciences des données dans le cadre d'un projet tutoré.

Il propose des fonctions simples et pédagogiques pour l'analyse de séries chronologiques et la réalisation de tests d'hypoyhèses en R.

L' objectif est d'accompagner les étudiants dans l'apprentissage des statistiques et de fournir un outil réutilisable pour les promos futures.

## Installation
 Le package peut etre installé directement depuis Github : 
 ```{r}
# install.packages("devtools")
devtools::install_github("nelson-fots/PTUTS/fonctions2A")
```
